Joystick Control for Cocos2d Objective-C (in Swift)
===================================================

I wanted to be able to pull a feed from a Wordpress blog.

Based off of a tutorial, I constructed an updated, performance-based architecture
to pull a JSON feed and display in a tableview with the help of Alamofire.

Enjoy!

===================================================

This project runs on Swift 3 and Xcode 8.3.3

This project is based on tutorial:
https://wlcdesigns.com/2015/07/create-a-wordpress-latest-posts-ios-app-in-swift/
